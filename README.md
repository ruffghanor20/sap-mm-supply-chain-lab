# SAP MM Supply Chain Lab
projeto GitHub completo focado em SAP MM + Supply Chain Analytics,A ideia é simular os processos do SAP MM (Materials Management) usando Python + SQL + dashboards.
Simulação de processos SAP Materials Management utilizando Python e análise de dados.

Este projeto recria os principais processos do SAP MM para estudo e portfólio profissional.

## Processos simulados

- Material Master
- Purchase Requisition
- Purchase Order
- Goods Receipt
- Inventory Management
- ABC Inventory Classification
- Supply Chain Analytics

## Tecnologias

Python  
Pandas  
SQL  
Power BI  

## Estrutura de processos

Procure to Pay:

Requisition → Purchase Order → Goods Receipt → Invoice → Inventory Update

## Objetivo

Demonstrar conhecimento funcional de SAP MM sem acesso a um ambiente SAP real.

Este projeto simula fluxos usados em empresas que utilizam SAP S/4HANA.
